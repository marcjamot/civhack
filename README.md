# Civhack

A multiplayer hacking simulation game.

## Client

Web-based client written in Typescript.

Libraries:

* Ecsy (https://ecsy.io/)
* PixiJS (https://www.pixijs.com/)
* Svelte (https://svelte.dev/)

**Build and run**

* `npm install`
* `npm run dev -- --open`

## Server

Server written in Golang.
